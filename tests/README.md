# Self-Leveling 测试文档

## 测试概览

本文档描述了 self-leveling 项目的测试策略、测试用例和执行方法。

### 测试类型

1. **单元测试 (Unit Tests)**
   - 使用 Vitest + React Testing Library
   - 位置: `tests/unit/`
   - 执行: `npm run test`

2. **端到端测试 (E2E Tests)**
   - 使用 Playwright
   - 位置: `tests/e2e/`
   - 执行: `npm run test:e2e`

3. **集成测试 (Integration Tests)**
   - API 端点测试
   - 位置: `tests/integration/`

## 执行测试

### 安装依赖

```bash
npm install
```

### 运行单元测试

```bash
npm run test
```

### 运行单元测试 (单次)

```bash
npm run test:run
```

### 运行测试覆盖率

```bash
npm run test:coverage
```

### 运行 E2E 测试

```bash
# 确保开发服务器运行中
npm run dev

# 运行 E2E 测试
npm run test:e2e
```

### 运行所有测试

```bash
npm test
npm run test:e2e
```
